# Servicio SEO Local — Pamplona

Carpeta de arranque para montar nuestro servicio productizado de SEO local (Google Business + reseñas + SEO local web) para negocios de cita previa en Pamplona.

## Cómo usar esto
1. Abre esta carpeta como proyecto en **Claude Code**.
2. Claude Code leerá `CLAUDE.md` automáticamente (es el contexto maestro).
3. Empieza por la **prioridad 1** del `CLAUDE.md`: el generador de auditorías.

10 audits gratis a clínicas/estudios de Pamplona → enviar como gancho → cerrar 2-3 antes del verano, con todo montado para gestionar en remoto.
